# yafpgatetris
Yet Another Tetris on FPGA Implementation

## About
Tetris game made on Terasic DE1-SoC board.

Input: PS/2 keyboard

Output: VGA display (1280x1024)

See also: http://habrahabr.ru/post/247535/
Video: http://www.youtube.com/watch?v=I6k_7Kwuilw
